# Vibe Check

Vibe Check is a rebellious wellness brand that rejects clinical jargon and "guilty" pleasures in favor of pure, unadulterated vibes.

## Getting Started

To view the website, simply open `index.html` in your web browser.

## Project Structure

- `index.html`: The main landing page, built with Tailwind CSS and Epilogue, Be Vietnam Pro, and Space Grotesk fonts.
